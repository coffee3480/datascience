вспомогательные таблицы - csv файлы, содержащие список имен, фамилий и названий
городов, для генерирования случайных данных, которыми мы будем заполнять базу 
данных

postgres_generating_data - питон код, который генерирует случайные данные, чтобы
заполнить базу данных. Создаёт csv файлы booking_passports, booking_rooms,
bookings, guests, passports. Для корректной работы программы необходимо изменить
переменую dirpath, в которой хранятся файлы.

bookings_extract_python - питон код, который выводит информацию о бронированиях
из базы данных

output.txt - текстовый файл, в который выводится информация о бронированиях из
базы данных

sql commands.txt - содержит все sql команды для создания БД, таблиц, заполнения данных, 
создания изображений(views) и индексов БД и вывода данных в файл.
