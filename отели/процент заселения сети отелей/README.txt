Файл data.csv содержит исходные данные об уровне заселения и дохода по дням.

price list.py - программа, которая формирует цены на 2025 год и выводит их в файл price list.xlsx

кореляция заселение-доход вычисляет коэффициент кореляции уровня заселения и 
дохода отелей за все время при помощи функции подгонки библиотеки scikit-learn
и QR-разложения

Кор Заселение-доход по годам вычисляет кореляцию между различными столбцами
файла data.xlsx по годам с 2023 по 2025

В папке графики по годам - графики уровня заселения по годам с 2017-го года
