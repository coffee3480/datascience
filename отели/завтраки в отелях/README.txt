Имеются данные о бронированиях в отелях. Данные представлены в виде эксель 
файлов. Необходимо распечатать список номеров, которые оплатили завтрак и
могут завтракать утром. Для этого распакуем эксель из зип файла, удалим
ненужные столбцы, для удобства персонала отсортируем таблицу в порядке
возрастания номера комнаты. У "СуперОтеля" 2 отдельных файла, которые
необходимо объединить в один.

В папке files bckp хранятся копии изначальных файлов, чтобы программу можно
было запустить заново, так как после успешного выполнения программы файлы 
будут изменены. 

Чтобы программа работала необходимо положить исходные файлы в одну папку с
программой.
