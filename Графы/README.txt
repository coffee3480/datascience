Программа принимает на вход текстовый файл:
на первой строчке два числа n и m - количество узлов и рёбер графа соответственно
на второй строчке через пробел записаны веса узлов по порядку
дальнейшие строчки описывают рёбра графа, указывая 2 связанных узла

Пример ввода:

3 3
20 15 40
1 2
1 3
2 3

Данный ввод описывает треугольный граф из 3-ёх узлов и 3-ёх рёбер,соединённых между собой.
Весы узлов составляют 20, 15 и 40 соответственно.

Каждый узел можно посетить лишь один раз, каждым ребром можно воспользоваться лишь один раз.
Необходимо вывести на экран максимальное значение суммы весов, которое можно получить при обходе графа.

Есть некоторое скопление островов, на каждом острове зарыт клад пиратов определенной ценности.
Ценность сокровищей задаётся числом. Каждый остров можно посетить лишь 1 раз, каждым маршрутом
можно воспользоваться 1 раз. Какая максимальная ценность сокровищей, которые можно вывезти?
